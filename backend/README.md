1. Create a .env file in the root directory and add the following env vars with appropriate values:
JWT_SECRET
PORT
DB_NAME
DB_HOST
DB_USER
DB_PASSWORD
NYLAS_CALLBACK_URL
NYLAS_CLIENT_ID
NYLAS_CLIENT_SECRET
NYLAS_REDIRECT_URL
CLOUDINARY_KEY
CLOUDINARY_SECRET
CLOUDINARY_CLOUD_NAME
2. Run "yarn install"
3. Run "yarn start"
